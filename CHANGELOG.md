# 변경 이력

이 파일의 각 버전 섹션(`## [버전]`)이 GitHub Release 의 패치노트로 자동 반영됩니다.
새 버전을 낼 때 맨 위에 해당 버전 섹션을 추가하고 VERSION 파일을 올리면 됩니다.

## [0.6.2] - 2026-07-10
### 수정
- 주간 할일 목록 빈칸 우클릭 시 붙여넣기 메뉴가 표시되도록 수정

## [0.6.1] - 2026-07-10
### 수정
- 할일을 캐릭터 위로 드래그해 올릴 때 삭제 사운드가 미리 재생되던 문제 수정

## [0.6.0] - 2026-07-10
### 추가
- 설정에 상황별 캐릭터 사운드 탭 추가
- 상황별 사운드 파일, 개별 음량, 1회/반복 재생 설정 추가
- 전체 사운드 사용 여부와 전체 음량 설정 추가
- 상황별 사운드 번들 파일로 wav/flac 지원

### 개선
- 연속 할일 추가 등 같은 리액션이 반복될 때 이미지와 사운드를 즉시 처음부터 다시 재생하도록 개선
- 캐릭터 클릭으로 목록을 열 때 open 리액션과 사운드가 클릭 시점에 시작되도록 개선
- 상황별 사운드 항목을 접고 펼칠 수 있도록 설정 화면 정리
- 리소스 폴더를 이미지(`resources/img`)와 사운드(`resources/sound`)로 분리
- 잠금 빌드에서 이미지와 사운드를 함께 암호화 팩에 포함하도록 개선

### 수정
- 사운드 설정 변경 시 이전 오디오 객체가 누적될 수 있는 문제 수정
- 음량 슬라이더 드래그 중 사운드 플레이어가 과도하게 다시 로드되던 동작 개선
