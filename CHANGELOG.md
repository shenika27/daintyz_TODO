# 변경 이력

이 파일의 각 버전 섹션(`## [버전]`)이 GitHub Release 의 패치노트로 자동 반영됩니다.
새 버전을 낼 때 맨 위에 해당 버전 섹션을 추가하고 VERSION 파일을 올리면 됩니다.

## [0.5.11] - 2026-07-05
### 추가
- 할일 입력창에서 새 할일의 중요도를 없음, 낮음, 중간, 높음 중 선택할 수 있는 기능 추가
- 할일 목록에서 중요도 아이콘 클릭으로 없음, 낮음, 중간, 높음 순서의 중요도 순환 기능 추가
- 할일 항목 우클릭 메뉴에서 중요도를 직접 선택할 수 있는 기능 추가

## [0.5.10] - 2026-07-03
### 개선
- 사이드 그리드 공통 헤더 규격을 통일해 밀린할일, 완료한 일, 타이머 타이틀 정렬 개선
- 캐릭터 클릭으로 그리드를 다시 열 때 할일 목록과 사이드 패널이 함께 자연스럽게 표시되도록 개선
- 사이드 패널이 먼저 그려졌다가 다시 애니메이션되는 깜빡임 완화

## [0.5.9] - 2026-07-03
### 추가
- 밀린할일 패널에서 완료한 일 목록으로 전환해 조회할 수 있는 기능 추가
- 완료한 일 표시 방식을 날짜별 보기와 상세 보기 중 선택할 수 있는 설정 추가
- 완료한 일 항목 또는 날짜를 오늘의 미완료 할일로 복제하는 메뉴 추가

### 개선
- 완료한 일 상세 항목 클릭 시 해당 날짜로 이동하고 대상 항목을 강조 표시하도록 개선
- 완료한 일 목록을 새로고침하거나 항목을 클릭해도 스크롤 위치가 유지되도록 개선
- 밀린할일 패널의 제목, 개수 표시, 전환 아이콘, 하단 버튼 표시 방식을 정리
- 설정 화면의 동작 항목 순서와 밀린할일 자동 이월 체크박스 배치를 정리

### 개선
