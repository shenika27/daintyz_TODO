# 변경 이력

이 파일의 각 버전 섹션(`## [버전]`)이 GitHub Release 의 패치노트로 자동 반영됩니다.
새 버전을 낼 때 맨 위에 해당 버전 섹션을 추가하고 VERSION 파일을 올리면 됩니다.

## [0.5.0] - 2026-07-01
### 추가
- 밀린할일 패널에서 항목 좌클릭/우클릭 시 날짜 보기, 완료 처리, 오늘로 옮기기 메뉴 표시
- 밀린할일 패널 하단에 모두 완료, 모두 오늘로 버튼 추가
- 할일 항목 우클릭 메뉴에 편집 기능 추가

### 개선
- 설정 화면의 일반 탭 내용을 동작, 화면, 관리 탭으로 재구성
- 미완료 할일 자동 이월 설정을 제거하고, 밀린할일 패널에서 필요한 항목만 직접 정리하는 흐름으로 변경
- 항상 위 해제 상태에서 그리드나 패널을 클릭하면 캐릭터도 함께 전면으로 올라오도록 개선
- 업데이트 확인에서 최신 버전, 새 버전, 확인 실패 상태를 구분해 안내
- 밀린할일 완료/이동 및 할일 텍스트 편집을 되돌리기 대상에 포함

### 수정
- 주간 화면에서 밀린할일 패널 항목을 우클릭하면 일간 화면으로 돌아가던 문제 수정
